Privacy Policy for World Geography Quiz App

World Geography Quiz app does not collect any user's information.
User does not need any account to use this android app.
User does not need any internet connection to use this application.
User just touch to select a choice of test section or best answer.
User can touch to see quiz report which will be cleared when exit app.
There are total 350 built-in multiple choice questions of geography knowledge.
User can touch CLEAR REPORT button to start a new quiz.
When user exits this app, all quiz answers and reports are cleared.
It is absolutely safe to use this app. User's privacy is absolutely protected.
